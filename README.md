# 4Wheels - Car Rental Prototype

Simple car rental prototype based on Oracle database schema.

## Live Site

https://fostersql.github.io/4Wheels/

## Features

- Browse available cars
- Filter by type
- Search by model/brand
- Simple booking system
- Responsive design

## Database Schema

Based on Oracle Express with tables:

- Users
- Car_Types
- Cars
- Rentals
- Payments
- Reviews

## Setup

1. Clone: `git clone https://github.com/FosterSQL/4Wheels.git`
2. Open `index.html` in browser

## Deploy to GitHub Pages

```bash
git add .
git commit -m "Update"
git push origin main
```

Enable GitHub Pages in repository Settings → Pages → Deploy from main branch.
